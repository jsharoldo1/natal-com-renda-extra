# **App Name**: Receitas de Natal Lucrativas

## Core Features:

- Landing Page Estratégica: Página de vendas com design estratégico focado na conversão de visitantes em clientes para o curso de confeitaria natalina.
- Semelhança com TudoGostoso: Imitação do layout do site TudoGostoso para criar familiaridade e confiança nos usuários.
- Vídeo de Vendas (VSL): Incorporação de um vídeo de vendas (VSL) persuasivo para apresentar o curso e seus benefícios.
- Seção de Comentários: Implementação de uma seção de comentários para depoimentos e prova social, aumentando a credibilidade do curso. Simulated user testimonials enhance the perception of real-world approval.
- Chatbot de Suporte: Implementação de um chatbot de suporte para responder dúvidas e fornecer informações adicionais sobre o curso.

## Style Guidelines:

- Esquema de cores claro e festivo, com foco em tons de vermelho cereja (#E53935) para remeter ao Natal.
- Cor de fundo suave e acolhedora, como um creme claro (#F5F5DC) para criar uma atmosfera convidativa.
- Cor de destaque vibrante e apetitosa, como um dourado suave (#FFD700) para evocar a riqueza dos sabores natalinos.
- Fonte para títulos: 'Playfair', serifada, elegante para destacar títulos e informações importantes.
- Fonte para corpo do texto: 'PT Sans', sans-serif, moderna e legível para garantir conforto na leitura.
- Ícones temáticos de Natal, como árvores, estrelas e presentes, para reforçar o clima festivo da landing page.
- Design responsivo para garantir que a landing page seja visualizada corretamente em dispositivos de todos os tamanhos.
- Animações sutis, como efeitos de fade-in e transições suaves, para tornar a experiência do usuário mais agradável e interativa.
